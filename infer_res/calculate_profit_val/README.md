# 股票交易利润计算工具（仅使用测试集日期）

基于LSTM模型预测结果，模拟交易策略并计算利润。**仅在测试集日期内使用模型预测，其他日期保持持有状态**，避免数据泄露问题。

## 功能说明

- 根据模型预测结果进行买入/卖出决策（仅在测试集日期内）
- 如果预测第二天涨（prediction=1），买入或持有
- 如果预测第二天跌（prediction=0），卖出
- 全仓操作（买入时使用全部现金，卖出时卖出全部持仓）
- **仅在测试集日期内使用模型预测，其他日期保持持有状态**
- 支持自定义交易日数量、起始日期和初始资金

## 使用方法

```bash
python calculate_profit.py <训练日志目录> <测试集CSV文件> [选项]
```

**注意：运行一次即可完成利润计算和可视化图表生成！**

### 参数说明

- `log_folder`: 训练日志目录路径（必需）
  - 例如: `logs/20251225_175236_train_518880_20251225`
  - 或绝对路径: `/home/wxl/wxlcode/stock/AKPRE_LSTM/logs/20251225_175236_train_518880_20251225`

- `val_csv`: 测试集CSV文件路径（必需）
  - 例如: `data/processed/NI0_20260104/val.csv`
  - 或绝对路径: `/home/ayaka/wxl/stock/try4/AKPRE_LSTM/data/processed/NI0_20260104/val.csv`

### 选项

- `--days`: 交易日数量，默认30天
- `--start-date`: 起始日期（格式: YYYY-MM-DD），如果不指定则从最后一天往前推
- `--capital`: 初始资金，默认10000元
- `--confidence-threshold`: 买入置信度阈值，只有当probability_up > threshold时才买入，默认0.5
- `--no-visualize`: 不自动生成可视化图表（默认会自动生成）

## 使用示例

```bash
# 使用默认参数（30个交易日，从最后一天往前推，初始资金10000元）
# 会自动生成所有结果文件和可视化图表
python calculate_profit.py /home/wxl/wxlcode/stock/AKPRE_LSTM/logs/20251225_175236_train_518880_20251225 \
    /home/ayaka/wxl/stock/try4/AKPRE_LSTM/data/processed/NI0_20260104/val.csv

# 指定交易日数量为60天
python calculate_profit.py logs/20251225_175236_train_518880_20251225 \
    data/processed/NI0_20260104/val.csv --days 60

# 指定起始日期和初始资金
python calculate_profit.py logs/20251225_175236_train_518880_20251225 \
    data/processed/NI0_20260104/val.csv --start-date 2024-11-01 --capital 20000

# 使用相对路径
python calculate_profit.py ../../logs/20251225_175236_train_518880_20251225 \
    ../../data/processed/NI0_20260104/val.csv --days 30

# 不自动生成可视化图表（只计算利润）
python calculate_profit.py logs/20251225_175236_train_518880_20251225 \
    data/processed/NI0_20260104/val.csv --no-visualize
```

## 输出结果

脚本会自动生成以下文件（保存在 `<股票码>_<投入金额>_<日期年月起止>` 文件夹下）：

1. **JSON结果文件** (`profit_result_<股票代码>_<时间戳>.json`)
   - 包含完整的计算结果、预测记录和交易记录
   - 包含每日的收盘价和实际涨跌情况（用于计算准确率）
   - 包含测试集日期信息

2. **CSV交易记录文件** (`trade_records_<股票代码>_<时间戳>.csv`)
   - 包含每日的交易操作记录，方便分析
   - 标记了哪些日期是测试集日期

3. **可视化图表**（自动生成，除非使用 `--no-visualize` 参数）
   - **金价涨跌曲线** (`*_price_changes.png`)
     - 收盘价走势曲线
     - 每日涨跌百分比柱状图
   - **资产对比曲线** (`*_asset_comparison.png`)
     - 交易策略总资产 vs 一直持有策略总资产
     - 显示两种策略的收益对比
   - **准确率收敛曲线** (`*_accuracy_convergence.png`)
     - 预测准确率随日期增加的累积准确率曲线
     - 滚动准确率曲线（窗口大小=10天）
   - **综合图表** (`*_combined.png`)
     - 包含以上三个图表的综合视图

## 结果说明

- **初始资金**: 开始时的资金
- **最终价值**: 结束时的总资产（现金+持仓价值）
- **利润**: 最终价值 - 初始资金
- **收益率**: (利润 / 初始资金) × 100%
- **交易次数**: 买入和卖出的总次数

## 重要特性

### 避免数据泄露

本工具的核心特性是**仅在测试集日期内使用模型预测**，其他日期保持持有状态。这样可以：

1. **避免数据泄露问题**：不会在训练集日期上使用模型进行交易决策
2. **客观评估模型性能**：只在真正的测试集上评估模型表现
3. **公平比较**：在非测试集日期保持持有状态，确保比较的公平性

### 交易策略说明

- **测试集日期**：使用模型预测进行买入/卖出决策
- **非测试集日期**：
  - 如果当前持仓，继续持有
  - 如果当前空仓，在第一个非测试集日期买入并持有（如果是从测试集卖出后进入非测试集日期）

## 注意事项

1. 交易日数量指的是实际的交易日，不包括周末和节假日
2. 如果指定了起始日期，会从该日期开始到数据的最新日期
3. 如果没有指定起始日期，会从最后一天往前推指定数量的交易日
4. 最后一天如果还有持仓，会以收盘价自动卖出
5. 测试集CSV文件必须包含"起始日期"列，工具会自动计算目标日期（起始日期+序列长度）

## 与 calculate_profit 的区别

| 特性 | calculate_profit | calculate_profit_val |
|------|------------------|---------------------|
| 数据泄露风险 | 可能包含训练集日期 | 仅使用测试集日期 |
| 测试集文件 | 不需要 | 必需 |
| 非测试集日期策略 | 使用模型预测 | 保持持有 |
| 评估客观性 | 可能不客观 | 客观 |
